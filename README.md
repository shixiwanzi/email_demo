# email_demo
java发送邮件示例(涵盖常用邮件服务器,邮件正文+附件)

1)springboot项目：
  添加依赖
		<dependency>
			<groupId>org.springframework.boot</groupId>
			<artifactId>spring-boot-starter-mail</artifactId>
		</dependency>
 2)spring项目：
  添加依赖
  <dependency>
      <groupId>javax.mail</groupId>
      <artifactId>mail</artifactId>
      <version>1.5.0-b01</version>
  </dependency>
